            1..Palindrome Checker
            
            Objective: Write a Python function is_palindrome(s) that checks whether a string s
 is a palindrome (reads the same backward as forward).
            

            2..Fibonacci Sequence Generator
            Objective: Write a Python function fibonacci(n) that generates the first n numbers of the Fibonacci sequence.
            

            3..Count Vowels and Consonants
            Objective: Write a Python function count_vowels_consonants(s) that counts the number of vowels and consonants in a string s.
            

            4..Prime Number Checker
            Objective: Write a Python function is_prime(n) that checks whether a given integer n is a prime number.
            

            5..Temperature Converter
            Objective: Write a Python program that converts a temperature from Celsius to Fahrenheit and vice versa.
            

            6..Count Word Frequency
            Objective: Write a Python function count_word_frequency(text) that counts the frequency of each word in a string text.
            

            7..Sorting a List of Tuples
            Objective: Write a Python function sort_tuples(tuples_list) that sorts a list of tuples based 
on the second element of each tuple.
            

            8..Sum of Digits in a Number
            Objective: Write a Python function sum_of_digits(n) that calculates the sum of the digits of an integer n.
            

            9..Find the Largest Element in a List
            Objective: Write a Python function find_largest(numbers) that finds the largest number in a list numbers.
            

            10..File Handling (Read and Write)
            Objective: Write a Python program that reads the contents of a text file and counts
