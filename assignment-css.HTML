            CSS Grid Layout for Image Gallery


                        5..Task: Create a simple image gallery using CSS Grid:

                                
  • The gallery should have 3 columns with equal spacing between the images.
    • 
                                
  • The images should be responsive, with each image adjusting its size based on the screen width.
    • 
                                
  • Use grid-template-columns to create the 3-column layout and grid-gap for spacing.
    • 
                                
  • On small screens, the images should stack into a single column.
